# SecretBoard
This is my semi-project for private board

## Summary

- 이 프로그램의 컨셉은 게시판형 개인 일기장이다
- DB 커넥팅과 계정관리 및 회원관리, Swing 으로 로그인 및 게시판 기능 구현 등을 연습하고자 만든다

## 사용 도구

- Oracle 11g xe
- java 8
- UI: Swing
- Eclipse Photon
